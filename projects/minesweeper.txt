Sometimes, it's important to reinvent the wheel. 

This project was notable for the following reasons:
    -   Implementation of MVC architecture and Observer/Observable design patterns
    -   Functional user interface
    -   Implementation of recursive logic for gameplay
    -   Written in Java, utilizing the Java Swing API to create user interface
